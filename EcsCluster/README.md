# AWS ECS Cluster

Built from a combination of the [`ecs-alb-service-task` Cloud Posse module](https://registry.terraform.io/modules/cloudposse/ecs-alb-service-task/aws/0.74.0)
and the [`ecs-container-definition` Cloud Posse module](https://registry.terraform.io/modules/cloudposse/ecs-container-definition/aws/0.61.1).

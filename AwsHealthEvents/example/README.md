# AWS Health Events Module Example

## How To Run and Test the Example

1) Initialize:

    ```sh
    terraform init
    ```

1) Plan

    ```sh
    terraform plan
    ```

1) Apply

    ```sh
    terraform apply
    ```

## How To Destroy the Example

1) Destroy the resources

    ```sh
    terraform destroy
    ```
